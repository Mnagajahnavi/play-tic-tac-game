# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built with clean UI and intuitive logic.

## ✨ Features

- Two-player gameplay (Player X and Player O)
- Visual win detection and game-over notification
- Reset functionality
- Responsive and minimal design

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
